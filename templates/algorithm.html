The Project

This project investigates lower-body injuries and concussions sustained in the NFL over sets of 2 seasons in order to attempt to determine factors associated with such injuries. We hope to provide insights to the factors correlating with high risk to protect players while maintaining the game as much as is safely possible.

×

The Process

  • Get the Data - Data acquisition and storage
  • Explore the Playing Field - What are we working with?
  • Clean it - Remove missing fields, fix issues, eliminate problems
  • Transform it - Prepare data for graphical and machine learning analyses
  • Establish machine learning models
  • Analyze it - What was learned? How can we improve the model?
  • Next Play - Improve the model, re-analyze
  • Present the results

    The Features

    This project features numerous types of media to explore the injuries in the NFL:

  • The Replays - interactive football field that allows the user to explore injurious plays
  • The Stats - interactive charts that allow the user to filter conditions and plays
  • Recent Injuries - shows for most recent injuries that have made headlines
  • Findings Ticker - Scrollable Top Bar to Examine Findings

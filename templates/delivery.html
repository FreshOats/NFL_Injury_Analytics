The Project

This project investigates lower-body injuries and concussions sustained in the NFL over sets of 2 seasons in order to attempt to determine factors associated with such injuries. We hope to provide insights to the factors correlating with high risk to protect players while maintaining the game as much as is safely possible.

    The Features

    This project features numerous types of media to explore the injuries in the NFL:

  • The Plays - interactive football field that allows the user to explore injurious plays
  • The Replays - interactive charts that allow the user to filter conditions and plays
  • Recent Injuries - shows for most recent injuries that have made headlines
